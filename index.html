Story Fumble

Description:
Story Fumble is an application where the user can blindly enter words into blank spaces creating a funny story. The user first chooses their language and category and then fills in words for the missing parts of speech in the story. After that, the user gets to choose a funny voice that reads the generated story out loud.


Technologies Used:
HTML, CSS, JavaScipt (ES6), React.js, JSX, Firebase, Web Speech API, Express, Node.js, MongoDB/Atlas, Mongoose, Bootstrap,and React Bootstrap.

SMSOL - Tutor Tracker

Description:
The Student Management System for Online Learning (SMSOL) is an application to help with scheduling students. The teacher or tutor can select a student and drag it on to the calendar to the chosen time slot. Appointments can be changed, moved, and deleted. Students can also be added and deleted from a master list.


Technologies Used:
UExpress-Handlebars, Express, Node.js, Sequelize, FullCalendar.io, Moment.js, Bootstrap, JQuery, Javascript, HTML, and CSS
